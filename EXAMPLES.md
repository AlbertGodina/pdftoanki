## Consells per Obtenir Millors Resultats

### 1. Prepara el PDF
- Assegura't que el text sigui llegible i ben estructurat
- Evita PDFs amb molt de contingut visual sense text
- Si és possible, utilitza PDFs amb capítols o seccions clares

### 2. Selecciona les opcions adequades
- Per matèries descriptives (història, biologia): activa totes les opcions
- Per matèries tècniques (programació, matemàtiques): centra't en Q&A i cloze
- Per memorització de definicions: activa definicions i llistes

### 3. Ajusta el nombre de flashcards
- 10-15 flashcards: documents curts (5-10 pàgines)
- 20-30 flashcards: documents mitjans (10-20 pàgines)
- 40-50 flashcards: documents llargs (20+ pàgines)
- Nota: més flashcards = més cost d'API

### 4. Revisa les flashcards generades
- Comprova que les respostes siguin correctes
- Verifica que les preguntes siguin clares
- Si alguna no és adequada, pots regenerar amb diferents opcions

### 5. Importa a Anki
- Descarrega l'arxiu .apkg
- Obre Anki
- Arxiu → Importar
- Selecciona l'arxiu .apkg
- Les flashcards apareixeran en un nou mazell "PDF Flashcards"

## Casos d'Ús Recomanats

### ✅ Ideal per a:
- Apunts de classe en format PDF
- Llibres de text digitals
- Articles acadèmics
- Manuals tècnics
- Documents de repàs
- Presentacions convertides a PDF

### ⚠️ No recomanat per a:
- PDFs escanejats sense OCR
- Documents amb molt de contingut visual (diagrames, gràfics)
- Llibres complets molt llargs (millor fer-ho per capítols)
- PDFs protegits amb contrasenya
- Documents amb taules complexes

## Preguntes Freqüents

**P: Puc editar les flashcards abans de descarregar?**
R: Actualment no, però és una funcionalitat planificada per futures versions. De moment, pots editar-les directament a Anki després d'importar-les.

**P: Per què algunes flashcards es repeteixen?**
R: Pot passar si el PDF té contingut repetit o molt similar. Pots eliminar els duplicats a Anki.

**P: Puc generar més flashcards del mateix PDF?**
R: Sí! Fes clic a "Generar altres flashcards" i obtindràs un nou conjunt diferent.

**P: Funciona amb PDFs en altres idiomes?**
R: Sí, però està optimitzat per català i espanyol. Per altres idiomes, els resultats poden variar.
